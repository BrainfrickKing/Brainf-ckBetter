ArnoldC

An imperative language where keywords are replaced with quotes from Arnold Schwarzenegger movies.

Befunge

A two-dimensional, stack-based language where the instruction pointer can move in any cardinal direction.

Brainf*ck

A minimalist language operating on a tape of memory cells with only eight simple commands.

JunLang

An esoteric language inspired by ArnoldC, with commands based on affectionate nicknames.

Thue

A non-deterministic, string-rewriting language where execution consists of applying substitution rules.
